# allotmint-mcp
Standalone MCP server for AllotMint (Spring Boot 4.1.0 + Java 25, MCP Java SDK) — Spring Boot MCP server for AllotMint API + local file access

## Contributing

Code style is enforced by [Spotless](https://github.com/diffplug/spotless) (Google Java Format). Before committing, run:

```
./mvnw spotless:apply
```

`mvn verify` fails the build if the code isn't formatted.
